# Code of Conduct

## Respect

Treat everyone with respect. Constructive criticism, technical debate, and code review are encouraged. Personal attacks, harassment, discrimination, and abusive behavior are not.

## Collaboration

Keep discussions focused on improving Aevros. Be open to feedback, explain design decisions clearly, and help contributors learn.

## Reporting

If you encounter unacceptable behavior, contact the project maintainer through GitHub or email.

## Enforcement

Maintainers may edit, lock, or remove comments, issues, pull requests, or users whose behavior harms the project or its community.
